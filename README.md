# Plant Detection 

## Overview

Plant Detection using YOLOv8 is a computer vision project that detects plants from images and videos using the YOLOv8 object detection model. The project is designed for agricultural monitoring and can be extended for real-time plant detection applications.

## Features

* Plant detection using YOLOv8
* Image and video inference
* Object tracking support
* Easy to modify and extend

## Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* Jupyter Notebook

## Project Structure

```text
├── YOLOv8_plant_detection.ipynb
├── track.py
├── requirements.txt
├── apple.mp4
└── suist/
```

## Installation

1. Clone the repository.
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Open `YOLOv8_plant_detection.ipynb` to train or test the model.
4. Run `track.py` to perform detection on images or videos.
